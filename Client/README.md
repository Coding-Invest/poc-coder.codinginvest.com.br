# Client

Este é o projeto Client, uma API Gateway ASP.NET Core que serve como intermediário entre o frontend e o backend.

## Funcionalidades
- Hospeda o frontend Angular.
- Expõe endpoints para integração com IA via MCP.
- Gerencia autenticação e comunicação.

## Como executar
1. Navegue para a pasta Client.
2. Execute `dotnet run`.

## Dependências
- ASP.NET Core 8.0
- Outros projetos: Domain

Para mais detalhes, consulte o README principal.
