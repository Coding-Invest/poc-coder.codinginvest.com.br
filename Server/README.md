# Server

Este é o projeto Server, implementando o servidor MCP (Model Context Protocol) para automação de tarefas via JSON-RPC.

## Funcionalidades
- Ferramentas para leitura/escrita de arquivos.
- Operações Git.
- Integração com IA generativa.

## Como executar
1. Navegue para a pasta Server.
2. Execute `dotnet run`.

## Dependências
- ASP.NET Core 8.0
- Outros projetos: Domain

Para mais detalhes, consulte o README principal.
