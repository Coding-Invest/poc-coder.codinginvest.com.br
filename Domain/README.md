# Domain

Este é o projeto Domain, uma biblioteca de classes compartilhadas definindo protocolos MCP e entidades comuns.

## Funcionalidades
- Definições de Tools, ProtocolRequest, etc.
- Lógica compartilhada entre Client e Server.

## Como usar
- Referenciado pelos projetos Client e Server.

## Dependências
- .NET 8.0

Para mais detalhes, consulte o README principal.
