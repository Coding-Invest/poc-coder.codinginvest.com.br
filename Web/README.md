# Web

Este é o projeto Web, uma aplicação frontend Angular para a interface de chatbot.

## Funcionalidades
- Chat em tempo real.
- Histórico de mensagens.
- Integração com backend via API.

## Como executar
1. Navegue para a pasta Web.
2. Execute `npm install`.
3. Execute `ng serve`.

## Dependências
- Angular 20
- RxJS, TypeScript

Para mais detalhes, consulte o README principal.
